# LLM 评估系统

本项目是一个完整的大语言模型（LLM）评估系统，包含前端界面和后端API服务。

## 项目结构

```
├── BackEnd/     # 后端服务 (Spring Boot)
└── FrontEnd/    # 前端界面 (Vue.js)
```

## 快速开始

### 后端启动
```bash
cd BackEnd
./gradlew bootRun
```

### 前端启动
```bash
cd FrontEnd
npm install
npm run dev
```
